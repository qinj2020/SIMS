查一下如何使用Git，一般习惯添加READ.md，LICENSE，.gitignore文件。
LICENSE文件从别人的commit下载修改，填写自己的姓名，时间
.gitignore文件指明无需上传的文件和子文件夹
例如：

/dirName/
text.jpg

在GitHub上复制命令到Git，根据GitHub提示输入用户名，密码， 完成上传
